# NeonCapture
## Prerequisites:
  - Install pypylon
  ```
    sudo apt-get update
    sudo apt-get upgrade
    sudo apt-get install python3-pip
    python3 -m pip install --upgrade pip
    pip3 install pypylon
  ```
## Run Sample
### Capture and Preview
